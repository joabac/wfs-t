# Introduction #

Este sera un aplicativo en modalidad **prototipo** que permitirá explorar las capacidades de diferentes servicios , API's y servidores de mapas, de código libre.

**Entre ellos:**

  * GeoServer
  * OpenLayers
  * GeoExt
  * Apache Tomcat
  * Apache HTTP



# Detalles #

Se busca con este aplicativo explorar a fondo las capacidades brindadas por los diferentes elementos nombrados, y sentar las bases para una futura ampliación y mejora progresiva de las capacidades brindadas.
